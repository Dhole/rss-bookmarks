# rss-bookmarks
Little service to store bookmars and generate RSS feeds to read on the go

```
Usage of rss-bookmarks:
  -assets string
    	Assets directory
  -data string
    	Data directory
  -port int
    	Server port (default 8080)
  -prefix string
    	URL HTTP server prefix
  -torify
    	Torify HTTP client requests
```
